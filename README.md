# innovateFPGA
Intel FPGA创新大赛项目，来自中华区决赛团队PR034，项目内容为基于友晶DE10-Nano开发板的依声辨位系统（Beamforming Positioning on DE10-Nano）。

本项目是一个从硬件设计到软件实现到数据显示的完整系统，实现了对给定声源的定位。项目利用了FPGA +ARM的组合，通过两块开发板分别进行数据的采集与处理、数据运算的软件控制。项目算法在并行优化等方面具有创新性。项目的优势在于FPGA在数据采集与处理，以及并行运算上的优秀表现，加上ARM在C语言软件实现的优异性能，在本项目中得到了极大的发挥。 我们设计的利用波束成形算法实现的定位算法，需要进行较大量的并行运算，因此在FPGA下的运算，性能可以得到极大的提升，这也是我们利用DE10-nano、DE0-soc 开发板进行此项目的初衷。

项目源代码包括Intel Quartus专案、信号采集板（server）源代码、数据运算板（client）源代码、平面展示源代码。# innovateFPGA-pr034
